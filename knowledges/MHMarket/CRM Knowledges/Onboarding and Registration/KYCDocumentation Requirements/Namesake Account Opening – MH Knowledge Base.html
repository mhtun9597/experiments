    Namesake Account Opening

    Willow (OB)
    March 03, 2026 08:38 Updated

    To outline the correct procedure for opening a namesake account while maintaining compliance, based on client type and hierarchy. 

    The Step-by-step guide on how to open a same name trading account in the CRM can be found here: How to open an extra trade account (namesake) for the clients?

    This article outlines the process for opening an account when a request is submitted through Zendesk. Depending on the type of requester, several scenarios may apply:
    1. Situation: Request received from Sales directly.
    2. Situation: Request received from IB
    3. Situation: Request received from Direct Client (DC)

    The Process Flowchart

    Situation 1: Request received from Sales directly

    1. Check whether the client has Rebate structure Set up
      • If a rebate scheme is already set up → proceed with opening the account.
        (Refer to: “Method for Checking the IB's Rebate (OB SOP)”)
      • If a rebate scheme is not set up → return the ticket to CS and request the missing information.
        Use Macro: OB1. More inquiries from CS
    2. Once Sales confirm the rebate scheme > proceed to opening an account.
      (Refer to: How to open an extra trade account (namesake) for clients?)
    3. Record the details in the AOR file.
      • Select the correct monthly file from the designated location.
    4. Reply to the ticket:
      • Use Macro OB2. Ticket is Solved, Send back to Agent to notify client,
      • Include the trading account number in the response.
    Do NOT modify “Leverage Type” or “Leverage.”

    Situation 2: Request received from IB

    1. Check whether the IB has the requested rebate scheme and promotion link set up
      • If a rebate scheme AND promotion link are already set up → proceed with opening the account.
        (Refer to: “Method for Checking the IB's Rebate (OB SOP)”)
      • If a rebate scheme AND promotion link are not set up → return the ticket to CS and request IB to contact their Account Manager (AM) for more information.
        Use Macro: OB1. More inquiries from CS
    Example:

    Client with Standard+3 requests ECN. In that case Advise client to contact their Account Manager. Account Manager will coordinate with IB. IB may share promotion link for client to open the account.

    Direct opening may affect IB’s business.
    1. Once Sales confirm the rebate scheme > proceed to opening an account.
      (Refer to: How to open an extra trade account (namesake) for clients?)
    2. Record the details in the AOR file.
      1. Select the correct monthly file from the designated location.
    3. Reply to the ticket:
      • Use Macro OB2. Ticket is Solved, Send back to Agent to notify client,
      • Include the trading account number in the response.
    4. Check IB Rebate Setup
      Refer to “Method for Checking the IB's Rebate (OB SOP)” for instructions.

    Situation 3: Request received from Direct Client (DC)

    1. Check whether the DC has account of the requested type already opened
      • If client is requesting an additional account of the type that he already has → proceed with opening the account.
      • If the client is asking to open an account type that is new for them → return the ticket to CS and advise to contact their Account Manager (AM) for more information.
        Use Macro: OB1. More inquiries from CS
    Important:
    Even if an IB has an ECN promotion link and rebate, do not open a new ECN account directly for a client who currently does not have ECN. AM/IB coordination is required.

      (Refer to: How to open an extra trade account (namesake) for clients?)
    2. Record the details in the AOR file.
      1. Select the correct monthly file from the designated location.
    3. Reply to the ticket:
